
## Project Proposal

This project uses Dynamic Difficulty Adjustment (DDA) and a Large Language Model (LLM), such as OpenAI, to generate and control opponent Pokémon teams.

The player uses their own team of 6 Pokémon. After each battle, the system adjusts the difficulty based on whether the player won and how many moves the battle took. The next opponent team is generated by the LLM based on the updated difficulty level.

Higher difficulty creates stronger, more strategic Pokémon teams. Lower difficulty creates simpler and more forgiving teams.

## Current Features

- Adjusts difficulty after each battle
- Generates a 6-Pokémon opponent team using an LLM
- Changes team strength based on difficulty percentage
- Includes Pokémon roles, items, abilities, natures, EVs, and moves

## How Difficulty Works CURRENTLY 

If the player wins, the difficulty increases.

If the player loses, the difficulty decreases.

If the battle takes fewer moves, the difficulty changes more strongly.

If the battle takes more than 80 moves, only a small base adjustment is applied.
